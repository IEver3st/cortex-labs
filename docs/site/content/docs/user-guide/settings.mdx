---
title: Settings
description: System, Viewer, Shortcuts, Design, and Experimental settings.
---

Open Settings from the top bar (gear icon).

## System

- **UI scale**: scales the interface via a CSS variable (`--es-ui-scale`).
- **Recents**: show/hide recent workspaces on the Home screen.

## Viewer

- **Exterior only default**
- **Window/glass targeting behavior**
- **WASD camera**
- **Grid visibility**
- **Light intensity / glossiness** defaults

## Shortcuts

Customize all key actions (new tabs, file selection, viewer toggles).

## Design

- Background color
- Body color

These settings are preview aids for readability and iteration.

## Experimental

Beta switches and diagnostics. If something behaves strangely after enabling experimental options, try disabling them and restarting.

## Persistence

Settings are saved locally (no cloud, no accounts).
