---
title: Hotkeys
description: Default shortcuts and how to customize them.
---

Hotkeys can be edited in **Settings → Shortcuts**.

## Default hotkeys

| Action | Default |
| --- | --- |
| Toggle Exterior Only | Alt + E |
| New Livery Tab | Alt + 1 |
| New All Tab | Alt + 2 |
| New EUP Tab | Alt + 3 |
| New Multi Tab | Alt + 4 |
| Toggle Control Panel | \\ |
| Select Model | Ctrl + O |
| Select Livery/Texture | Ctrl + L |
| Select Glass/Window | Ctrl + G |
| Toggle Multi Gizmo | Alt + G |
| Swap Selected Slot (Multi) | Alt + X |

## Tips

- If a single key doesn’t “stick”, add a modifier (Ctrl/Alt/Shift). Cortex Studio rejects some unmodified keys to avoid interfering with typing.
- On macOS-style keyboards, Ctrl bindings may map via Meta in the browser layer.
